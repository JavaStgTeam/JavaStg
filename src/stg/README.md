# stg 包

## 功能描述
stg 包是 JavaStg 游戏引擎的核心包，包含了游戏的基础架构、游戏逻辑和工具类。该包是整个游戏引擎的主要组成部分，提供了从底层渲染到上层游戏逻辑的完整实现。

## 包结构
stg 包包含以下子包：

### 1. stg.base
基础组件包，提供窗口管理、键盘输入处理等基础功能。

### 2. stg.game
游戏核心逻辑包，包含以下子包：
- `bullet`：子弹相关类
- `enemy`：敌人相关类
- `event`：游戏事件类
- `item`：道具相关类
- `laser`：激光相关类
- `obj`：游戏对象基类
- `player`：玩家相关类
- `stage`：关卡相关类
- `ui`：游戏界面相关类

### 3. stg.util
工具类包，提供数学计算、资源管理、事件总线等工具功能。

## 核心功能
1. **游戏循环**：通过 `GameLoop` 类实现游戏的主循环
2. **碰撞检测**：通过 `CollisionSystem` 类实现游戏对象间的碰撞检测
3. **游戏渲染**：通过 `GameRenderer` 类实现游戏画面的渲染
4. **游戏状态管理**：通过 `GameStateManager` 类管理游戏状态
5. **游戏世界**：通过 `GameWorld` 类管理游戏世界中的对象
6. **输入处理**：通过 `GameCanvas` 类处理玩家输入

## 设计理念
stg 包采用了模块化设计，将不同功能分离到不同的子包中，便于维护和扩展。同时，通过接口定义（如 `IGameObject`、`IGameWorld` 等）实现了组件间的解耦，提高了代码的可测试性和可扩展性。

## 依赖关系
- 依赖 Java 标准库中的 AWT 和 Swing 包用于界面渲染
- 内部子包之间存在依赖关系，如 `game` 包依赖 `base` 包和 `util` 包
